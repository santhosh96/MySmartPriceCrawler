# MySmartPriceCrawler
Spider for collecting data of smartphones of different brands from mysmartprice.com

## Spiders

### Location of spiders:

	Root
   	|-->Directory(BrandName)
   	    |-->Directory(BrandName)
   	    	|-->spiders
   	    		|-->mobile.py

### Example spider:
	
	Root
	|-->acer
		|-->acer
			|-->spiders
				|-->mobile.py
